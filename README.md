# ClearView Local Pros Demo Website

This is a simple static website built for practicing the GitHub to Netlify workflow.

## Files

- `index.html` controls the page content and layout.
- `styles.css` controls the colors, spacing, fonts, responsive layout, and card design.
- `script.js` controls small interactions like the mobile menu, current footer year, and practice quote button.
- `images/` stores the gallery/project images used by the website.

## Netlify settings for this project

Because this site uses plain HTML/CSS/JS:

- Build command: leave blank
- Publish directory: `/` or leave as the project root if Netlify detects it automatically

## Practice edits

Try these beginner updates after the site is live:

1. Add a new review inside the reviews section in `index.html`.
2. Upload new images into the `images` folder.
3. Add new gallery cards in the gallery section in `index.html`.
4. Commit the changes to GitHub.
5. Watch Netlify automatically redeploy the new version.
